Margin, borders and Padding... what are these?

With a range of features on CSS with the purpose of enhancing html coding, understanding the margin, border and padding of an element is a key component to master. Just like the placement of the image within this picture frame above. We'll come back to that, first lets define each factor.

Margin

Beginning with the outer factor of an element, the margin is what determines the conditions that are between the element and all other elements of the site. It is commonly used to reference the distance between elements.

In order to customize each side of the margin surrounding a specific dimension is applied to the object with the following code. This can also be simplified listed like the second featured code. It is key to note that the dimensions are to be listed in clockwise order.

Border

Moving inwards but not quite. The border, as the name suggest refers to the area directly around the element. We can apply a range of custom conditions listed below.

Padding

Further inside the element you will find the padding which is refers to the space between the content of an element and how it interacts with its edges.
